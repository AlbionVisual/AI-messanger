Первая инициализация:

Для бэкенда из папочки `Backend`:

```
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

Для фронтенда из `Ai_chat_frontend`:

```
npm install
```

После пулла:
Бэкенд:

```
pip install -r requirements.txt
```

Для запуска:
Фронт:

```
npm start
```

Бэк:

```
python app.py
```
